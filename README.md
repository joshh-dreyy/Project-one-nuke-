# My Web Project

A basic web project template with HTML, CSS, and JavaScript.

## Project Structure

```
project-root/
├── index.html          # Main HTML file
├── styles/
│   └── main.css       # Main stylesheet
├── scripts/
│   └── main.js        # Main JavaScript file
└── README.md          # This file
```

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Start editing the files to build your project

## Features

- Responsive design
- Clean and organized file structure
- Basic interactive button example
- Modern CSS with CSS variables
- Well-documented code

## Browser Support

This template works in all modern browsers including:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
